Perguntas Frequentes

O aplicativo é seguro?

Sim! O COTW Grind Counter Overlay é uma ferramenta legítima e segura. Como utilizamos atalhos globais (hotkeys) para que você possa contar abates sem sair do jogo e ao mesmo tempo que usa atalhos do jogo, alguns antivírus mais sensíveis podem gerar emitir um alerta de aplicativo suspeito, mas não se preocupe, logo em seguida após analisar o app ele emitirá outro alerta de aplicativo seguro. Estamos submetendo o app constantemente aos principais motores de segurança especializados (Microsoft Defender, Avast, etc.) para garantir a melhor experiência.

O que é a disponibilidade no WinGet?

Estar no WinGet (Repositório Oficial da Microsoft) significa que nosso instalador foi verificado e aprovado pelos padrões de segurança da Microsoft. Isso garante que você está baixando um software íntegro e confiável diretamente das ferramentas oficiais do Windows.

Por que o app precisa detectar as teclas do teclado?

Para que o contador funcione enquanto você está com a janela do jogo aberta, precisamos monitorar as teclas de atalho (como o Numpad + e -). Esse sistema só fica ativo durante uma sessão de grind e não registra nenhuma outra informação, garantindo sua privacidade.

O app é um rastreador automático?

Não. O Grind Counter é um contador manual assistido via overlay. Nós não injetamos código no jogo nem lemos a memória do processo, o que torna o uso 100% seguro contra banimentos (anti-cheat friendly).

Como instalar via WinGet?

Para usuários avançados que preferem o terminal, basta abrir o PowerShell e digitar:
winget install WeudesMarquez.COTWGrindCounterOverlay

Como faço para atualizar o aplicativo?

O aplicativo possui um sistema de atualização automática integrado. Sempre que lançarmos uma melhoria ou correção, você será notificado ao abrir o app. Se você instalou via WinGet, também pode atualizar pelo terminal usando winget upgrade WeudesMarquez.COTWGrindCounterOverlay.
